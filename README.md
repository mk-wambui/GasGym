# GasGym

A fully responsive multi-page gym business website built with HTML5, CSS3, JavaScript (ES6+), and Bootstrap 5.

## Pages
- Home
- About Us
- Classes
- Trainers
- Membership
- Gallery
- Blog
- Contact

## Technologies
- HTML5 & CSS3
- Bootstrap 5
- JavaScript ES6+
- Git & GitHub

## Features
- Responsive design (mobile, tablet, desktop)
- Dark/Light mode toggle with localStorage
- Class schedule filter by category (Cardio, Strength, Yoga, HIIT)
- Contact form with JavaScript validation

## Team
GasGym is run by a team of 6 certified trainers covering strength, cardio, yoga, nutrition, and boxing/kickboxing specialties.

## Design
Bold maroon and blush pink colour scheme with Bebas Neue and Inter typography, targeting a Nairobi-based fitness audience.

Deployed via GitHub Pages.
